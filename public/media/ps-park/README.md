# PS Park Photos

Поместите сюда фотографии для страницы PlayStation Park.

## Ожидаемые файлы

| Файл | Где используется |
|------|-----------------|
| `table-1.jpg` | Карточка стола 1 |
| `table-2.jpg` | Карточка стола 2 |
| `table-3.jpg` | Карточка стола 3 |
| `gallery-1.jpg` | Галерея — игровая зона |
| `gallery-2.jpg` | Галерея — PS5 крупный план |
| `gallery-3.jpg` | Галерея — атмосфера |
| `gallery-4.jpg` | Галерея — интерьер |

## Рекомендации

- Формат: JPEG или WebP
- Карточки столов (`table-N.jpg`): соотношение 4:3, минимум 800×600px
- Галерея (`gallery-N.jpg`): квадратные, минимум 600×600px
- Размер файла: до 300 KB каждый (используйте сжатие)
